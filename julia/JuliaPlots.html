Plots en Julia
Ulises M. Alvarez

La librera, Plots, es una interface de visualizacin y una caja de herramientas. Nos permite usar el poder de muchas de las libreras de grficos de Julia, i.e. motores, usando una sintaxis nica.

Como principales ventajas podemos destacar:

  • Dialecto de grficos universal. No importa el motor de grficos que estemos usando, PyPlot, Gadfly, Plotly, PGFPlots, etc, podemos usar un slo dialecto para construir las grficas.

  • Se puede cambiar el motor de grficos con un slo comando y no es necesario reescribir el cdigo.

  • Es muy sencillo agregar capas, --anotaciones, leyendas, incluso grficos adicionales superpuestos--, a un grfico.

Para saber ms: https://juliaplots.github.io/

Uso de Plots¶

Una vez instaladas la libreras, podemos proceder a usar Plots con el motor de nuesta preferencia. Es importante sealar que:

  • Para generar estas notas usamos el motor, GR.
  • Para sesiones o documentos interactivos, i.e. html, se recomienda el motor plotly().
  • Para poder exportar desde Jupyter o IJulia, es necesario especificar el formato de salida. Les recomendamos SVG o PNG. Ejemplo:
plot(rand(10), fmt = :svg)

URL: https://juliaplots.github.io/output/

In [23]:
using Plots  # Cargamos librera
gr()         # Definimos motor
Out[23]:
Plots.GRBackend()

Colores¶

Ahora, manipularemos colores, transparencia y agregaremos una capa de color superpuesta. Seguimos usando el motor PyPlot:

In [33]:
y       = rand(100)    # Cien nmeros aleatorios

#plotly()              # Pruebe diferentes motores!

plot(0:10:100,         # Iniciamos grfica
rand(11,4),            # Subconjuntos aleatorios
lab     = "lines",     # Texto de la leyenda
w       = 3,           # Ancho de las lneas
palette = :grays,      # Paleta de color
fill    = (0, :auto),  # rea con color
       = 0.4,         # Transparencia
fmt     = :svg)        # Opcin para generar las notas
